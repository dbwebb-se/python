for
==============================
Examples in Python 3


```python
"""
Loop through list of items
"""
my_list = [1,2,3]
for x in my_list:
	print(x)

my_list = ["one", "two", "three"]
for x in my_list:
	print(x)


"""
Loop through the characters in a string
"""
my_str = "Hello"
for x in my_str:
	print(x)


```
