Types
==============================
Examples in Python 3


```python
"""
Print out the type of some values
"""
# Numbers
print(type(42))
print(type(4.2))

# String
print(type("Hello world!"))

# Booleans
print(type(True))
print(type(False))

# None
print(type(None))

# Multiline
hello = """
Hello
World!
"""
print(type(hello))

# Function
def test():
	print("test-function")
print(type(test))


```
